Introduction to Deep Learning (CSE 490/599 G1)

Ali Farhadi, Daniel Gordon, Samuel Gehman, James Park, Satvik Shukla,
Aaron Walsman, Jifan Zhang
Fall 2019

Course Webpage Homework Code

I was the head TA and organizer for the Introduction to Deep Learning course at UW in Fall 2019. In this class we taught the basics of deep learning. I additionally created homeworks for the course in which students implement a neural network library using only Python, Numpy, and Numba. We additionally used Colab notebooks and PyTorch to train real models on GPUs. We had over 160 students! If other instructors are interested in using my solutions and tests, please email me.
