Deep Learning in Practice Seminar

I organized a seminar on using deep learning in practice. In it, we discussed how to apply modern techniques to solve real research problems. We also explored the effectiveness of various methods. More information, as well as slides, videos, and code are posted on the seminar website https://sites.google.com/cs.washington.edu/deeplearninginpractice/.

We went through various topics including:

  1. Activation Functions
  2. How to Choose a Deep Network
  3. Zero Shot Learning
  4. Generalization
  5. Translational Invariance
  6. Data Pipelines
  7. Deep Learning for Mobile
  8. Real-Time Object Detection
  9. Bayesian Hyperparameter Optimization
  10. Generative Adversarial Networks
  11. DQN in a Day
  12. Learning Interpretable Representations
  13. Fast Convolution Algorithms
  14. Recurrent Neural Networks
  15. Distributed Deep Learning
